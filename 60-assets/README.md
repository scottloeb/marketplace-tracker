# 🖼️ Static Assets

Images, icons, and other static files for the application.

## 📁 Directory Contents

- **browser-extension/** - Chrome extension for one-click marketplace capture
- **icons/** - Application icons and branding assets
- **images/** - Screenshots, logos, and visual assets

## 🧩 Browser Extension

The `browser-extension/` folder contains a complete Chrome extension that:
- **Automatically extracts** listing data from marketplace pages
- **Saves to Google Sheets** queue for processing 
- **Integrates perfectly** with existing automation pipeline
- **Works on** Craigslist, Facebook Marketplace, eBay, OfferUp

See `browser-extension/README.md` for installation and setup instructions.

## 🎯 Purpose

Static asset storage for UI enhancement and browser extension functionality.

**Dependencies**: None (static files only)
